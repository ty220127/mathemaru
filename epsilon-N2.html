[#8] 定義：数列の収束，極限

$\{a_n\}_{n\in\mathbb{N}}$ を実数列$\nr{collection-family}{223}$ とする．この $\{a_n\}$ に対して実数 $\alpha$ が存在して，任意の正の数 $\varepsilon$ に対して適当な番号 $N(\varepsilon)$ を定めると，それ以上の自然数 $n$ に対して $|a_n-\alpha|\lt\varepsilon$ が成り立つとき，このことを $\{a_n\}$ は $\alpha$ に【収束する (converge)】という：

\begin{align} {}^\exists \alpha\in\mathbb{R}\,\,\text{s.t.}\,{}^\forall \varepsilon \gt 0,\, {}^\exists N(\varepsilon)\in\mathbb{N}\,\, \text{s.t.}\ {}^\forall n\in \mathbb{N},\, \left[\,n\geq N(\varepsilon)\,\to\,|a_n-\alpha|<\varepsilon\right]. \end{align}

$\{a_n\}$ が $\alpha$ に収束することを

$\displaystyle\lim_{n\rightarrow\infty}a_n = \alpha\,$ 或いは $\,a_n\rightarrow\alpha\,(n\rightarrow\infty)$

などと書く．また，この $\alpha$ を $\{a_n\}$ の【極限 (limit)】或いは【極限値 (limit value)】と呼ぶ．


 

[#9] 例：収束する数列の例

$\limit_{n\rightarrow\infty}\fraction{1}{n} = 0$.

証明

$\varepsilon\gt 0$ を任意に固定し，それに対し $N(\varepsilon) = \lfloor \varepsilon^{-1} \rfloor +1$ とおく．このとき，$N(\varepsilon) \gt \varepsilon^{-1}$ である．ここで，$n\geq N(\varepsilon)$ なる自然数 $n$ に対して

\begin{align} \left|\fraction{1}{n}-0\right| = \fraction{1}{n} \leq \fraction{1}{N(\varepsilon)} \lt \varepsilon \end{align}

が成り立つ．すなわち，$\limit_{n\rightarrow\infty}\fraction{1}{n} = 0$ である．

補足

証明の検討過程としては，まず絶対値を $N(\varepsilon)$ の式で上から押さえることを考える．これにより

\begin{align} \left|\fraction{1}{n}-0\right| = \fraction{1}{n} \leq \fraction{1}{N(\varepsilon)} \end{align}

を得る．次に，その $N(\varepsilon)$ の式を $\varepsilon$ で上から押さえられるための，つまり

\begin{align} \fraction{1}{N(\varepsilon)} \lt \varepsilon \end{align}

が成立するための $N(\varepsilon)$ の必要条件を考え，それに基づき $N(\varepsilon)$ を決定する．今回は $N(\varepsilon)\gt \varepsilon^{-1}$ がそれに当たるので，それを満たす一例として $N(\varepsilon) = \lfloor \varepsilon^{-1} \rfloor +1$ を選んだ．


 

[#35] 例：収束する数列の例

$k$ を定数として，$\limit_{n\rightarrow\infty}\fraction{\,n\,}{\,n+k\,} = 1.$

証明

$\varepsilon\gt 0$ を任意に固定し，それに対し

\begin{align} N(\varepsilon) &= \max\left\{\left\lfloor \fraction{(1-\varepsilon)k}{\varepsilon} \right\rfloor +1,\ 0\right\}\\ &= \left\{\begin{array}{cc} \left\lfloor \fraction{(1-\varepsilon)k}{\varepsilon} \right\rfloor +1 & (0\lt \varepsilon\leq 1) \\ 0 & (1\lt \varepsilon)\end{array}\right. \end{align}

とおく．ここで，$n\geq N(\varepsilon)$ なる自然数 $n$ に対して

\begin{align} \left|\fraction{n}{n+k}-1\right| = \left|-\fraction{k}{n+k}\right| = \fraction{k}{n+k}\leq \fraction{k}{N(\varepsilon)+k}. \end{align}

$0\lt \varepsilon\leq 1$ に対して，

\begin{align} \left|\fraction{n}{n+k}-1\right| &\leq \fraction{k}{N(\varepsilon)+k} = \fraction{k}{\left\lfloor \fraction{(1-\varepsilon)k}{\varepsilon} \right\rfloor+1+k}\\ &\lt \fraction{\varepsilon k}{(1-\varepsilon)k +\varepsilon k} =\varepsilon. \end{align}

$\varepsilon \gt 1$ に対して，

\begin{align} \left|\fraction{n}{n+k}-1\right| &\leq \fraction{k}{N(\varepsilon)+k} = \fraction{k}{0+k} = 1\lt \varepsilon.\\ \end{align}

よっていずれの場合においても $\limit_{n\rightarrow\infty}\fraction{n}{n+k} = 1.$


 

[#371] 命題：極限値の一意性

任意の数列 $\{a_n\}_{n\in\mathbb{N}}$ に対し，その極限は一意である：

\begin{align} \left(\lim_{n\to\infty}a_n= \alpha\right)\land \left(\lim_{n\to\infty}a_n= \beta\right) \ \Longleftrightarrow\ \alpha = \beta. \end{align}

証明

$\alpha \gt \beta$ とする．いま，$\varepsilon = (\alpha -\beta)/2$ とすれば，極限の定義$\nr{epsilon-N2}{8}$ より，ある自然数 $N_1,\ N_2$ が存在して，任意の $n\in\mathbb{N}$ に対して

\begin{align} &n\geq N_1 \ \Longrightarrow\ |a_n-\alpha |\lt \varepsilon,\\ &n\geq N_2 \ \Longrightarrow\ |a_n-\beta |\lt \varepsilon\\ \end{align}

が成り立つ．ここで，$N_{12} = \max\{N_1,\ N_2\}$ とおけば，

\begin{align} &n\geq N_{12} \ \Longrightarrow\ (|a_n-\alpha |\lt \varepsilon)\land (|a_n-\beta |\lt \varepsilon)\\ \end{align}

が従う．この下では更に

\begin{align} |a_n-\alpha| + |a_n-\beta | \lt 2\varepsilon = \alpha -\beta\\ \end{align}

が成り立つ．一方，三角不等式より

\begin{align} |a_n-\alpha| + |a_n-\beta | \gt |\alpha -a_n + a_n -\beta| = |\alpha -\beta|\\ \end{align}

が成り立つ．以上をまとめると

\begin{align} |\alpha -\beta|\lt \alpha -\beta\\ \end{align}

となるが，これは矛盾である．つまり $\alpha\lt \beta$ は成り立たない．
同様にして $\beta\lt \alpha$ も成立しないので，$\alpha = \beta$ である．


 

[#374] 命題：和の極限は極限の和

極限を持つ数列 $\{a_n\},\ \{b_n\}$ に対して

\begin{align} \lim_{n\to\infty} (a_n +b_n) = \lim_{n\to\infty}a_n + \lim_{n\to\infty}b_n. \end{align}

証明

$\lim_{n\to\infty} a_n = \alpha,\ \lim_{n\to\infty}b_n = \beta$ とする．このとき，極限の定義$\nr{epsilon-N2}{8}$ より，任意の $\eta \gt 0$ に対して $M_1(\eta),\ M_2(\eta)\in \mathbb{N}$ が存在して，任意の $n\in\mathbb{N}$ に対して

\begin{align} n\geq M_1(\eta)\ &\Longrightarrow\ |a_n-\alpha |\lt \eta,\\ n\geq M_2(\eta)\ &\Longrightarrow\ |b_n-\beta |\lt \eta \end{align}

が成り立つ．ここで，$\varepsilon\gt 0$ を任意に固定して，自然数 $N(\varepsilon)$　を

\begin{align} N(\varepsilon) = \max\left\{M_1\left(\fraction{\varepsilon}{2}\right),\ M_2\left(\fraction{\varepsilon}{2}\right)\right\} \end{align}

で定める．すると $n\geq N(\varepsilon)$ に対して $n\geq M_1(\varepsilon/2)$ かつ $n\geq M_2(\varepsilon/2)$ であるから，

\begin{align} |a_n-\alpha|\lt \fraction{\varepsilon}{2},\quad |b_n-\beta|\lt \fraction{\varepsilon}{2} \end{align}

が従う．よって，三角不等式を用いることで

\begin{align} |a_n+b_n -(\alpha +\beta)| &= |a_n-\alpha +b_n -\beta|\\ &\leq |a_n-\alpha| + |b_n -\beta|\\ &\lt \fraction{\varepsilon}{2} + \fraction{\varepsilon}{2} = \varepsilon \end{align}

を得ることから，題意は示される．


 

[#36] 命題：定数倍の極限は極限の定数倍

極限を持つ数列 $\{a_n\}$ と定数 $c$ に対して

\begin{align} \lim_{n\to\infty}ca_n = c\lim_{n\to\infty}a_n. \end{align}

証明

$\lim_{n\to\infty} a_n = \alpha$ とする．このとき，極限の定義$\nr{epsilon-N2}{8}$ より，任意の $\eta \gt 0$ に対して $M(\eta)\in \mathbb{N}$ が存在して，任意の $n\in\mathbb{N}$ に対して

\begin{align} n\geq M(\eta)\ &\Longrightarrow\ |ca_n-c\alpha |\lt \eta \end{align}

が成り立つ．ここで $\varepsilon \gt 0$ を任意に固定し，自然数 $N$ を

\begin{align} N(\varepsilon) = M\left(\fraction{\varepsilon}{|c|}\right) \end{align}

で定める．このとき，$n\geq N(\varepsilon)$ に対して

\begin{align} |ca_n-c\alpha| &= |c||a_n-\alpha|\leq |c|\fraction{\varepsilon}{|c|} = \varepsilon \end{align}

を得ることから，題意が示される．

[#188] 定義：上界，下界，有界

数列 $\{a_n\}$ に対してある実数 $B$ が存在して，

${}^\forall n\in \mathbb{N},\,\,\,a_n\leq B$

を満たすことを $\{a_n\}$ は【上に有界 (bounded from above)】であるといい，この $B$ を【上界 (upper bound)】と呼ぶ．

また，数列 $\{a_n\}$ に対してある実数 $b$ が存在して，

${}^\forall n\in \mathbb{N},\,\,\,a_n\geq b$

を満たすことを $\{a_n\}$ は【下に有界 (bounded from below)】であるといい，この $b$ を【下界 (lower bound)】と呼ぶ．

更に，$\{a_n\}$ が上に有界でかつ下に有界であることを $\{a_n\}$ は【有界 (bounded)】という．


 

[#372] 命題：有界と極限

数列 $\{a_n\}$ に対して，$b,\ B\in \mathbb{R}$ がそれぞれ $\{a_n\}$ の下界，上界であるならば，

\begin{align} b\leq \lim_{n\to\infty}a_n\leq B. \end{align}
