[#24] 定義：測度，測度空間

$(\varOmega,\,\mathcal{F})$ を可測空間とする．関数 $\mu:\mathcal{F}\rightarrow [0,\,\infty)$ が $(\varOmega,\,\mathcal{F})$ 上の【測度 (measure)】であるとは， $\mu$ が以下の条件を満たすことを云う：

  • $\mu(\emptyset) = 0$,
  • 無限加算個の部分集合列 $\{A_n\}_{n\in\mathbb{N}}$ について，
    $\{A_n\}\subset\mathcal{F}$ かつ $\{A_n\}$ が互いに素 [#196] $\,\,\Longrightarrow\,\,\mu\left(\bsqcu_{n=1}^{\infty} A_n\right) = \sig_{n=1}^{\infty}\mu(A_n)$
    が成り立つ【測度の $\sigma$ 加法性】．

また，組 $(\varOmega,\,\mathcal{F},\,\mu)$ を【測度空間 (measure space)】と呼ぶ．


 

[#25] 命題：測度の有限加法性

有限個の集合列 $\{A_n\}_{n=1}^{N}$ について

$\{A_n\}\subset\mathcal{F}$ かつ $\{A_n\}$ が互いに素 [#196] $\,\,\Longrightarrow\,\,\mu\left(\bsqcu_{n=1}^{N} A_n\right) = \sig_{n=1}^{N}\mu(A_n).$

証明

$n= N+1,\,N+2,\,\cdots$ に対して $A_n=\emptyset$ と定めれば $\{A_n\}_{n\in\mathbb{N}}$ は互いに素な $\mathcal{F}$ の部分集合列であり，

$\mu\left(\bcu_{n=1}^{N} A_n\right)=\mu\left(\bcu_{n\in\mathbb{N}} A_n\right).$

一方，[#24]-i より $\mu(\emptyset)=0$ であるから，

$\sig_{n\in\mathbb{N}}\mu(A_n) = \sig_{n=1}^{N}\mu(A_n).$

以上と測度の $\sigma$ 加法性 [#24]-ii より，題意は示される．


 

[#205] 命題：集合を用いた測度の分解

集合 $A,\,B\in \mathcal{F}$ に対して

$\mu(B) = \mu(A\cap B) + \mu(B\setminus A).$

特に，$A\subset B$ ならば

$\mu(B) = \mu(A) + \mu(B\setminus A).$

証明

$B= (A\cap B)\sqcup(B\setminus A)$ であるから$\nr{difference-set}{55}$，測度の有限加法性$\nr{measure}{25}$ より

$\mu(B) = \mu(A\cap B) + \mu(B\setminus A).$

また，$A\subset B$ の下で $A\cap B = A$ であるから$\nr{set}{57}$，このとき

$\mu(B) = \mu(A) + \mu(B\setminus A).$


 

[#26] 命題：測度の単調性

$A,\,B\in\mathcal{F}$ について，$A\subset B$ ならば

$\mu(A)\leq \mu(B).$

証明

$A\subset B$ なる $A,\ B\in\mathcal{F}$ に対して

$\mu(B)-\mu(A)=\mu(B\setminus A)$

が成り立つが$\nr{measure}{205}$，測度の値域$\nr{measure}{24}$ より $\mu(B\setminus A)\geq 0,\ $ つまり $\mu(B)-\mu(A)\geq 0$ である．よって題意を得る．


 

[#46] 命題：和の測度と積の測度の関係

測度空間 $(\varOmega,\,\mathcal{F},\,\mu)$ 上の可測集合　$A,\,B\in\mathcal{F}$ に対して，

$\mu(A\cup B) = \mu(A) + \mu(B) -\mu(A\cap B).$

証明

集合 $A,\ B$ に対して $A\cup B = A \cup (B\setminus A)$ であるから$\nr{difference-set}{97}$，測度の有限加法性$\nr{measure}{25}$ より

\begin{align} \mu(A\cup B) = \mu(A)+\mu(B\setminus A). \tag{$*$} \end{align}

更に

$\mu(B\setminus A) = \mu(B) - \mu(A\cap B).$

が成り立つことから$\nr{measure}{205}$，これを $(*)$ 式に代入することで所望の結果を得る．


 

[#206] 命題：集合列を用いた測度の分解

集合 $A\in \mathcal{F}$ とする．

  • $\{B_n\}_{n=1}^{\infty}\subset\mathcal{F}$ が互いに素で $\sqcup_{n=1}^{\infty}B_n = \varOmega$ を満たすとき

    \begin{align*} \mu(A) = \sum_{n=1}^{\infty}\mu(A\cap B_n) \end{align*}

  • $\{B_n\}_{n=1}^{N}\subset\mathcal{F}$ が互いに素で $\sqcup_{n=1}^{N}B_n = \varOmega$ を満たすとき

    \begin{align*} \mu(A) = \sum_{n=1}^{N}\mu(A\cap B_n) \end{align*}

証明

前者について示す．
$m,\,n\lt \infty$ に対して

\begin{align*} (A\cap B_m)\cap (A\cap B_n) = A\cap B_m\cap B_n = \emptyset \end{align*}

であるから，$\{A\cap B_n\}_{n=1}^{\infty}$ は互いに素である．また，

\begin{align*} \bsqcu_{n=1}^{\infty}(A\cap B_n) \refeq{set}{131} A\cap \bsqcu_{n=1}^{\infty}B_n = A\cap \varOmega = A \end{align*}

が成り立つから，測度の $\sigma$ 加法性 [#24] より

\begin{align*} \mu(A) = \mu\left(\bsqcu_{n=1}^{\infty}(A\cap B_n)\right)= \sum_{n=1}^{\infty}\mu(A\cap B_n) \end{align*}

後者についても，測度の有限加法性 [#25] を用いることで同様に示せる．


 

[#89] 命題：測度の劣加法性

必ずしも互いに素でない集合の列 $\{A_n\}_{n\in\mathbb{N}}$ に対して

$\{A_n\}\subset\mathcal{F}\,\,\Longrightarrow\,\, \mu\left(\bcu_{n\in\mathbb{N}}A_n\right) \leq \sig_{n\in\mathbb{N}}\mu(A_n).$

証明

集合列 $\{B_n\}_{n\in\mathbb{N}}$ を

$B_1=A_1,\quad B_n = A_n\setminus \bcu_{k=1}^{n-1}A_k\,(n\geq 2)$

と定義する．このとき，$\{B_n\}$ は互いに素でかつ $\bcu_{n\in\mathbb{N}}B_n = \bcu_{n\in\mathbb{N}}A_n$ であるから

$\mu\left(\bcu_{n\in\mathbb{N}}A_n\right)=\mu\left(\bcu_{n\in\mathbb{N}}B_n\right)=\sig_{n\in\mathbb{N}}\mu(B_n).$

一方，$B_n$ の定義より $B_n\supset A_n$ であるから，測度の単調性 [#26] より $\mu(B_n)\leq \mu(A_n)$ が成り立つので

$\sig_{n\in\mathbb{N}}\mu(B_n)\leq \sig_{n\in\mathbb{N}}\mu(A_n).$

以上の議論をまとめることにより，題意は示される．


 

[#27] 命題：測度の上方連続性

集合の列 $\{A_n\}_{n\in\mathbb{N}}\subset\mathcal{F}$ が 単調非減少列$\nr{limit-set}{190}$ であるならば，

$\limit_{n\rightarrow\infty}\mu(A_n) = \mu\left(\lim_{n\rightarrow\infty}A_n\right).$

証明

集合列 $\{B_n\}_{n\in\mathbb{N}}$ を

$B_1=A_1,\quad B_n = A_n\setminus A_{n-1}\,(n\geq 2)$

と定義する．このとき，$\{B_n\}$ は互いに素でかつ $\bcu_{n\in\mathbb{N}}B_n = \bcu_{n\in\mathbb{N}}A_n$ であるから

$\mu\left(\bcu_{n\in\mathbb{N}}A_n\right)=\mu\left(\bcu_{n\in\mathbb{N}}B_n\right)=\sig_{n\in\mathbb{N}}\mu(B_n).$

一方 $A_n=\bcu_{k=1}^{n}B_k$ より $\mu(A_n) = \mu\left(\bcu_{k=1}^{n}B_k\right) = \sig_{k=1}^{n}\mu(B_k)$ であるから，

$\limit_{n\rightarrow\infty}\mu(A_n) = \limit_{n\rightarrow\infty}\sig_{k=1}^{n}\mu(B_k) = \sig_{k\in\mathbb{N}}\mu(B_k).$

以上の議論をまとめることにより，題意は示される．


 

[#28] 命題：測度の下方連続性

集合の列 $\{A_n\}_{n\in\mathbb{N}}\subset\mathcal{F}$ が単調非増加列$\nr{limit-set}{190}$ であるならば，

$\limit_{n\rightarrow\infty}\mu(A_n) = \mu\left(\lim_{n\rightarrow\infty}A_n\right).$

証明

集合列 $\{B_i\}_{i\in\mathbb{N}}$ を $B_i = A_1\setminus A_i$ をおくと，$\{B_i\}$ は $\mathcal{F}$ の部分集合列で単調増大列になるから， 測度の上方連続性$\nr{measure}{27}$ と単調性$\nr{measure}{26}$ より

$\begin{align*}\mu\left(\bcu_{i\in\mathbb{N}}B_i\right) &= \bcu_{i\in\mathbb{N}}\mu(B_i) \\ &= \limit_{i\rightarrow\infty}(\mu(A_1)- \mu(A_i)) \\ &= \mu(A_1) - \limit_{i\rightarrow\infty}\mu(A_i).\end{align*}$

一方，$\bcu_{i\in\mathbb{N}}B_i$ について，de Morgan の法則$\nr{collection-family}{132}$ より

$\begin{align*}\bcu_{i\in\mathbb{N}}B_i &= \bcu_{i\in\mathbb{N}}(A_1\cap A_i^\cpl) = A_1 \cap \bcu_{i\in\mathbb{N}}A_i^\cpl\\ &= A_1 \cup \left(\bca_{i\in\mathbb{N}}A_i\right)^\cpl = A_1\setminus \bca_{i\in\mathbb{N}}A_i\end{align*}$

と計算できるから，単調性$\nr{measure}{26}$ より

$\begin{align*}\mu\left(\bcu_{i\in\mathbb{N}}B_i\right) &= \mu\left(A_1\setminus \bca_{i\in\mathbb{N}}A_i\right)\\ &= \mu(A_1) - \mu\left(\bca_{i\in\mathbb{N}}A_i\right).\end{align*}$

以上の結果をまとめると，

$\begin{align*} &\mu(A_1) - \limit_{i\rightarrow\infty}\mu(A_i) = \mu(A_1) - \mu\left(\bca_{i\in\mathbb{N}}A_i\right)\\ \Longleftrightarrow\,\,& \limit_{i\rightarrow\infty}\mu(A_i) = \mu\left(\bca_{i\in\mathbb{N}}A_i\right)\end{align*}$

を得る．


 

[#226] 命題：Borel(ボレル)-Cantelli(カンテリ) の補題 (Borel-Cantelli lemma)

測度空間 $(\varOmega,\,\mathcal{F},\,\mu)$ について，測度 $\mu$ は非負であるとする．このとき可測集合列 $\{A_n\}_{n=1}^{\infty}\subset \mathcal{F}$ に対して

\begin{align*} \sum_{n=1}^{\infty}\mu(A_n)\lt\infty\,\,\Longrightarrow\,\,\mu\left(\underset{n\rightarrow\infty}{\mathrm{lim\,sup}}\ A_n\right) = 0. \end{align*}

ここで $\underset{n\rightarrow\infty}{\mathrm{lim\,sup}}\ A_n$ は $\{A_n\}$ の上極限 [#189] ．

証明

集合列 $\{\bigcup_{m=n}^{\infty}A_n\}$ は単調非増加列 [#190] であるから

\begin{align*} \mu\left(\underset{n\rightarrow\infty}{\mathrm{lim\,sup}}\ A_n\right) &\refeq{limit-set}{190} \mu\left(\bca_{n=1}^{\infty}\bcu_{m=n}^{\infty}A_n\right) \refeq{limit-set}{191} \mu\left(\lim_{n\rightarrow\infty}\bcu_{m=n}^{\infty}A_n\right)\\ &\refeq{measure}{28} \lim_{n\rightarrow\infty}\mu\left(\bcu_{m=n}^{\infty}A_n\right) \refleq{measure}{89} \lim_{n\rightarrow\infty}\sum_{m=n}^{\infty}\mu(A_n)=0. \end{align*}

補足

この補題は，確率測度の議論で非常に重要となるが，一般の測度に対しても成立する．なお，確率論においては，ここから第二補題 [#227] も導かれる．


 

[#16] 演習：確率測度 → $\mathsf{[\#12]}$

正六面体のサイコロを1回投げたときに，ある値の目が出る「確率」を，以下で定義される測度空間 $(\varOmega,\,2^\varOmega,\,P)$ 上の測度 $P$ で定めることができる：

$\begin{align*} &\varOmega = \{1,\,2,\,3,\,4,\,5,\,6\},\ \ P(\varOmega)=1,\\ &{}^\forall \omega\in\varOmega,\ P(\{\omega\}) = p. \end{align*}$

ここで $\varOmega$ は出得る目の値，$ P(\{\omega\})$ は $\omega$ の目が出る確率を表す．

  1. $p$ の値を求めよ．
  2. 偶数の目が出る確率を求めよ．
  3. $E=\sig_{\omega =1}^{6}\omega P(\{\omega\})$ を求めよ．
解答
  1. $\sig_{\omega=1}^{6}P(\{\omega\}) \,\refeq{measure}{25}\, P(\varOmega)\,\,\Longleftrightarrow\,\, 6p=1\,\,\Longleftrightarrow\,\,p=\color{red}{\fraction{1}{6}}\,.$
  2. $P(\{2,\,4,\,6\}) \,\refeq{measure}{25}\, P(\{2\})+P(\{4\})+P(\{6\}) = 3p = \color{red}{\fraction{1}{2}}\,.$
  3. $E = \sig_{\omega =1}^{6}\omega P(\{\omega\}) \underset{(1)}{=} \fraction{1}{6}\sig_{\omega=1}^{6} \omega = \color{red}{\fraction{7}{2}}.$
