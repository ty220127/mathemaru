[#24]定義：測度，測度空間

$(\varOmega,\,\mathcal{F})$ を可測空間とする．関数 $\mu:\mathcal{F}\rightarrow [0,\,\infty)$ が $(\varOmega,\,\mathcal{F})$ 上の【測度 (measure)】であるとは， $\mu$ が以下の条件を満たすことを云う：

  • $\mu(\emptyset) = 0$,
  • $\{A_i\}_{i\in\mathbb{N}}\subset\mathcal{F}$ が任意の $i,\,j\in\mathbb{N}$ に対して $i\neq j$ ならば $A_i\cap A_j$ であるとき，$\mu\left(\bcu_{i\in\mathbb{N}} A_i\right) = \sig_{i\in\mathbb{N}}\mu(A_i)$（測度の $\sigma$ 加法性）.

また，組 $(\varOmega,\,\mathcal{F},\,\mu)$ を【測度空間 (measure space)】と呼ぶ．



[#25]命題：測度の有限加法性
$\{A_i\}_{i=1}^{n}\subset\mathcal{F}$ が任意の $i,\,j\in\mathbb{N}$ に対して $i\neq j$ ならば $A_i\cap A_j$ であるとき，$\mu\left(\bcu_{i=1}^{n} A_i\right) = \sig_{i=1}^{n}\mu(A_i)$.
証明 $i= n+1,\,n+2,\,\cdots$ に対して $A_i=\emptyset$ と定めれば，$\mu\left(\bcu_{i=1}^{n} A_i\right)=\mu\left(\bcu_{i\in\mathbb{N}} A_i\right)$． 一方，$\mu(\emptyset)=0$ であるから（[#24]-i），$\sig_{i\in\mathbb{N}}\mu(A_i) = \sig_{i=1}^{n}\mu(A_i)$． 以上と測度の $\sigma$ 加法性（[#24]-ii）より，題意は示される．


[#26]命題：測度の単調性
$A,\,B\in\mathcal{F}$ が $A\subset B$ ならば $\mu(B\backslash A) = \mu(B)-\mu(A)$ であり，それ故に $\mu(B)-\mu(A)\geq 0$ である.
証明 $A\cap B\backslash A = \emptyset$ かつ $A\cup B\backslash A = B$ であるから測度の有限加法性[#25]より，
$\mu(A)+\mu(B\backslash A) = \mu(B)\,\,\Longleftrightarrow\,\,\mu(B\backslash A) = \mu(B)-\mu(A)$.
また，測度の値域[#24]より，$\mu(B)-\mu(A) = \mu(B\backslash A)\geq 0$ である．


[#46]命題：和の測度と積の測度の関係
測度空間 $(\varOmega,\,\mathcal{F},\,\mu)$ 上の可測集合　$A,\,B\in\mathcal{F}$ に対して，$\mu(A\cup B) = \mu(A) + \mu(B) -\mu(A\cap B)$.
証明１ $A\cup B = A \cup (B\backslash A)$ であるから，測度の有限加法性 [#25] より $\mu(A\cup B) = \mu(A)+\mu(B\backslash A)$． 更に，$(A\cap B)\subset B$ から測度の単調性 [#26] より $\mu(B\backslash A) = \mu(B\backslash (A\cap B)) = \mu(B) - \mu(A\cap B)$． 以上により，$\mu(A\cup B) = \mu(A) + \mu(B) -\mu(A\cap B)$ が成り立つ．
証明２ $A\cup B = A \cup (B\backslash A)$ かつ $B=(B\backslash A) \cup (A\cap B)$ であるから，測度の有限加法性 [#25] より $\mu(A\cup B) +\mu(A\cap B) = \mu(A)+ \mu(B\backslash A) + \mu(A\cap B) = \mu(A) + \mu(B)$．よって $\mu(A\cup B) = \mu(A) + \mu(B) -\mu(A\cap B)$ が成り立つ．


[#27]命題：単調増大列と測度の連続性定理

集合の列 $\{A_i\}_{i\in\mathbb{N}}\subset\mathcal{F}$ が $A_1\subset A_2\subset \cdots$ を満たすならば，

$\limit_{i\rightarrow\infty}\mu(A_i) = \mu\left(\bcu_{i\in\mathbb{N}}A_i\right).$

証明

集合列 $\{B_i\}_{i\in\mathbb{N}}$ を

$B_1=A_1,\quad B_i = A_i\backslash A_{i-1}\,(i\geq 2)$

と定義する．このとき，$\{B_i\}$ は互いに素でかつ $\bcu_{i\in\mathbb{N}}B_i = \bcu_{i\in\mathbb{N}}A_i$ であるから

$\mu\left(\bcu_{i\in\mathbb{N}}A_i\right)=\mu\left(\bcu_{i\in\mathbb{N}}B_i\right)=\sig_{i\in\mathbb{N}}\mu(B_i).$

一方 $A_n=\bcu_{i=1}^{n}B_i$ より $\mu(A_n) = \mu\left(\bcu_{i=1}^{n}B_i\right) = \sig_{i=1}^{n}\mu(B_i)$ であるから，

$\limit_{n\rightarrow\infty}\mu(A_n) = \limit_{n\rightarrow\infty}\sig_{i=1}^{n}\mu(B_i) = \sig_{i\in\mathbb{N}}\mu(B_i).$

以上の議論をまとめることにより，題意は示される．



[#28]命題：単調減少列と測度の連続性定理

集合の列 $\{A_i\}_{i\in\mathbb{N}}\subset\mathcal{F}$ が $A_1\supset A_2\supset \cdots$ を満たし，かつ $\mu(A_1)\lt \infty$ であるならば，

$\limit_{n\rightarrow\infty}\mu(A_n) = \mu\left(\limit_{n\rightarrow\infty}A_n\right).$

証明

集合列 $\{B_i\}_{i\in\mathbb{N}}$ を $B_i = A_1\setminus A_i$ をおくと，$\{B_i\}$ は $\mathcal{F}$ の部分集合列で単調増大列になるから，単調増大列と測度の連続性定理 [#27] より

$\begin{align*}\mu\left(\bcu_{i\in\mathbb{N}}B_i\right) &= \bca_{i\in\mathbb{N}}\mu(B_i) \\ &= \limit_{i\rightarrow\infty}(\mu(A_1)- \mu(A_i)) \\ &= \mu(A_1) - \limit_{i\rightarrow\infty}\mu(A_i).\end{align*}$

一方，$\bcu_{i\in\mathbb{N}}B_i$ について，

$\begin{align*}\bcu_{i\in\mathbb{N}}B_i &= \bcu_{i\in\mathbb{N}}(A_1\cap A_i^\cpl) = A_1 \cap \bcu_{i\in\mathbb{N}}A_i^\cpl\\ &= A_1 \cup \left(\bca_{i\in\mathbb{N}}A_i\right)^\cpl = A_1\setminus \bca_{i\in\mathbb{N}}A_i\end{align*}$

と計算できるから，

$\begin{align*}\mu\left(\bcu_{i\in\mathbb{N}}B_i\right) &= \mu\left(A_1\setminus \bca_{i\in\mathbb{N}}A_i\right)\\ &= \mu(A_1) - \mu\left(\bca_{i\in\mathbb{N}}A_i\right).\end{align*}$

以上の結果をまとめると，

$\begin{align*} &\mu(A_1) - \limit_{i\rightarrow\infty}\mu(A_i) = \mu(A_1) - \mu\left(\bca_{i\in\mathbb{N}}A_i\right)\\ \Longleftrightarrow\,\,& \limit_{i\rightarrow\infty}\mu(A_i) = \mu\left(\bca_{i\in\mathbb{N}}A_i\right)\end{align*}$

を得る．
