[#416] 定義：行列指数関数

正方行列 $A$ に対し以下で与えられる行列 $\mathrm{e}^A$ を $A$ の【指数関数行列 (matrix exponential)】と呼ぶ：

\begin{align} \mathrm{e}^A = \sum_{k=0}^{\infty} \fraction{A^k}{k!} = I_n + A + \fraction{A^2}{2!} + \fraction{A^3}{3!} + \cdots. \end{align}

ただし $I_n$ は $n$ 次単位行列$\nr{matrix}{184}$ である．なお，$\mathrm{e}^A$ は $\exp A$ などと書くこともある．

補足

指数関数行列は，指数関数 $\mathrm{e}^x$ の Maclaurin 展開に形式的に正方行列 $A$ を代入したものである．


 

[#417] 例：零行列の指数関数

$(n,\, n)$ 型零行列$\nr{matrix}{178}$ $O_{nn}$ に対して以下が成り立つ：

\begin{align} \mathrm{e}^{O_{nn}} = I_n. \end{align}

証明

$k\geq 1$ に対して

\begin{align} \fraction{(O_{nn})^k}{k!} = O_{nn} \end{align}

であるから，

\begin{align} \mathrm{e}^{O_{nn}} \,\refeq{matrix-exponential}{416}\, I_n + \sum_{k=1}^{\infty} \fraction{(O_{nn})^k}{k!} \,\refeq{matrix}{179}\, I_n. \end{align}


 

[#418] 例：対角行列の指数関数

$n$ 次対角行列$\nr{diagonal-matrix}{286}$ $\DeclareMathOperator{\diag}{diag} D= \diag(d_1,\ d_2,\ \cdots,\ d_n)$ に対して，

\begin{align} \mathrm{e}^D = \diag(\mathrm{e}^{d_1},\ \mathrm{e}^{d_2},\ \cdots,\ \mathrm{e}^{d_n}). \end{align}

ただし，$D^0 = I_n$ ( $n$ 次単位行列）とする．

証明

\begin{align} \mathrm{e}^D &\,\refeq{matrix-exponential}{416}\, \sum_{k=0}^{\infty} \fraction{(\diag(d_1,\ d_2,\ \cdots,\ d_n))^k}{k!}\\ &\,\refeq{diagonal-matrix}{288}\, \sum_{k=0}^{\infty} \fraction{(\diag(d_1^k,\ d_2^k,\ \cdots,\ d_n^k))}{k!}\\ &\,=\, \diag\left( \sum_{k=0}^{\infty}\fraction{d_1^k}{k!},\ \sum_{k=0}^{\infty}\fraction{d_2^k}{k!},\ \cdots,\ \sum_{k=0}^{\infty}\fraction{d_n^k}{k!}\right)\\ &\,=\, \diag(\mathrm{e}^{d_1},\ \mathrm{e}^{d_2},\ \cdots,\ \mathrm{e}^{d_n}). \end{align}

[#429] 定義：行列指数関数の逆行列

任意の正方行列 $A$ に対して行列指数関数 $\mathrm{e}^A$ は正則$\nr{regular}{163}$ であり，

\begin{align} (\mathrm{e}^A)^{-1} = \mathrm{e}^{-A}. \end{align}

証明

任意の正方行列 $A$ に対して，$A$ と $-A$ は可換である：

\begin{align}A\cdot (-A) = (-A)\cdot A = -A^2.\end{align}

よって，これらの行列指数関数の積に関し以下が成り立つ（ただし $O,\ I$ はそれぞれ $A$ と同じサイズの零行列と単位行列）：

\begin{align} &\mathrm{e}^{A}\mathrm{e}^{-A}\,\refeq{matrix-exponential}{428}\, \mathrm{e}^{A-A} = \mathrm{e}^{O} \,\refeq{matrix-exponential}{417}\, I,\\ &\mathrm{e}^{-A}\mathrm{e}^{A}\,\refeq{matrix-exponential}{428}\, \mathrm{e}^{-A+A} = \mathrm{e}^{O} \,\refeq{matrix-exponential}{417}\, I. \end{align}

つまり $\mathrm{e}^{A}\mathrm{e}^{-A} = \mathrm{e}^{-A}\mathrm{e}^{A} = I$ であるから，$\mathrm{e}^{A}$ は正則であり，その逆行列は $\mathrm{e}^{-A}$ である$\nr{regular}{163}$．

[#427] 命題：対角化可能な行列の指数関数

正方行列 $A$ が正則行列$\nr{regular}{163}$ $P$ で対角化可能$\nr{diagonalization}{298}$ であり，その際の対角行列$\nr{diagonal-matrix}{286}$ が $\varLambda$ であるとき，$\mathrm{e}^A$ は $P$ で $\mathrm{e}^\varLambda$ に対角化される$\nr{matrix-exponential}{418}$．

証明

正方行列 $A$ に関する仮定より，$k\in\mathbb{N}$ に対して

\begin{align}A^k = P\varLambda^k P^{-1}\tag{$*$}\end{align}

が成り立つ$\nr{diagonalization}{301}$．また，任意の行列 $M$ に対して $M^0 = I$(単位行列) と仮定すれば

\begin{align}P\varLambda^0 P^{-1} = PP^{-1} \,\refeq{regular}{163}\, I = A^0\end{align}

であるから，$k\in \mathbb{N}\cup \{0\}$ に対して $(*)$ が成立する．よって

\begin{align} \mathrm{e}^A &= \sum_{k=0}^{\infty} \fraction{A^k}{k!} \,\refeq{matrix-exponential}{416}\, \sum_{k=0}^{\infty} \fraction{P\varLambda^k P^{-1}}{k!}\\ &= P\left( \sum_{k=0}^{\infty} \fraction{\varLambda^k}{k!}\right) P^{-1} = P\mathrm{e}^\varLambda P^{-1}. \end{align}


 

[#423] 命題：ユニタリ行列は Hermite 行列の指数関数
