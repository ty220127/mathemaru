[#48]定義：集合に関する諸定義

命題関数 [#75] $P(x)$ を真とするような変項 $x$ の集まりのことを【集合(set)】と呼び，

$\{x\,|\,P(x)\}$

と表す．$A$ を命題関数 $P(x)$ で定義された集合であるとするとき，記号 $a$ に対して $P(a)$ が真であることを 「$a$ は $A$ に属する ($a$ belongs to $A$)」と云い，$a\in A$ と表す．また，$a$ を集合 $A$ の【元 (要素; element)】と呼ぶ．



[#110]定義：空集合

変項の内容に関わらず恒偽命題であるような命題関数により定義される集合を【空集合(empty set)】と呼び，$\emptyset$ と書く．



[#111]定義：集合の包含
  • 集合 $A$，$B$ が，記号 $a$ に対して

    $a\in A\,\,\Longrightarrow\,\, a\in B$

    を満たすことを「$A$ は $B$ に含まれる ($A$ is included in $B$)」と云い，$A\subset B$ と表す．
  • $A\subset B$ かつ $B\subset A$ であること，つまり

    $a\in A\,\,\Longleftrightarrow\,\, a\in B$

    であることを「集合 $A$ と $B$ は等しい ($A$ and $B$ are equal)」と云い，$A=B$ と表す．
  • $A\subset B$ かつ $A\neg B$ であるような集合 $A$ を $B$ の【真部分集合 (proper subset)】と呼ぶ．


[#49]命題：集合は自身の部分集合

空集合 $\emptyset$ は任意の集合の部分集合である．

証明

任意の $x$ に対して $x\in\emptyset$ が成り立たないことから，含意の定義 [#65] よりすべての集合 $A$ に対して $[x\in\emptyset\Rightarrow x\in A]$ である．

補足

論理包含の命題に対して，仮定が偽である場合に結論の真偽に関わらずこれを真とすることで都合が良くなる例の一つである．



[#50]命題：集合は自身の部分集合

任意の集合 $A$ に対して，$A\subset A$.

証明

[#66] より任意の $x$ に対して $[x\in A \Rightarrow x\in A]$ である．



[#51]定義：集合の和と共通部分
  • 集合 $A$ と $B$ のうち一方或いはその両方に属するような要素の集合を $A$ と $B$ の【和(sum)】または【合併(union)】と呼び，$A\cup B$ と書く：

    $A\cup B= \{x\,|\,(x\in A) \lor (x\in B)\}.$

  • 集合 $A$ と $B$ の両方に属するような要素の集合を $A$ と $B$ の【共通部分(intersection)】または【交叉(meet)】と呼び，$A\cap B$ と書く：

    $A\cap B= \{x\,|\,(x\in A) \land (x\in B)\}.$



[#52]命題：集合演算の交換法則

集合 $A,\,B$ に対して

$A\cup B=B\cup A$，$A\cap B=B\cap A$.

証明

命題の交換法則 [#60] により従う．例えば $[A\cup B= B\cup A]$ に対して，

$\begin{align*} x\in A\cup B\,\,&\Longleftrightarrow\,\, (x\in A) \lor (x\in B)\\ &\Longleftrightarrow\,\,(x\in B) \lor (x\in A)\,\,\Longleftrightarrow\,\, x\in B\cup A.\end{align*}$



[#53]命題：集合演算の結合法則

集合 $A,\,B,\,C$ に対して

$\begin{align*}(A\cup B)\cup C &=A\cup (B\cup C),\\ (A\cap B)\cap C &=A\cap (B\cap C).\end{align*}$

証明

命題の結合法則 [#61] により従う．例えば $[(A\cup B)\cup C=A\cup (B\cup C)]$ に対して，

$\begin{align*} \,\,& x\in (A\cup B)\cup C\\ \Longleftrightarrow\,\,& (x\in A\cup B) \lor (x\in C)\\ \Longleftrightarrow\,\,& ((x\in A) \lor (x\in B)) \lor (x\in C)\\ \Longleftrightarrow\,\,& (x\in A) \lor ((x\in B) \lor (x\in C))\\ \Longleftrightarrow\,\,& (x\in A) \lor (x\in B\cup C)\\ \Longleftrightarrow\,\,& x\in A\cup (B\cup C).\end{align*}$

補足

この命題により，括弧付けによる 2 式の区別は基本的に不要であるため，強調等の意図がない場合は括弧を省略する：

$\begin{align*} (A\cup B)\cup C &=A\cup (B\cup C) = A\cup B\cup C,\\ (A\cap B)\cap C &=A\cap (B\cap C) = A\cap B\cap C.\end{align*}$



[#54]命題：集合演算の分配法則

集合 $A,\,B,\,C$ に対して

$\begin{align*}(A\cap B)\cup C &=(A\cup C)\cap (B\cup C),\\ (A\cup B)\cap C &=(A\cap C)\cup (B\cap C).\end{align*}$

証明

命題の分配法則 [#62] により従う．例えば $[(A\cap B)\cup C =(A\cup C)\cap (B\cup C)]$ に対して，

$\begin{align*}\,\,& x\in (A\cap B)\cup C\\ \Longleftrightarrow\,\,& (x\in A\cap B)\lor (x\in C)\\ \Longleftrightarrow\,\,& ((x\in A)\land (x\in B))\lor (x\in C)\\ \Longleftrightarrow\,\,& ((x\in A)\lor (x\in C))\land ((x\in B)\lor (x\in C))\\ \Longleftrightarrow\,\,& (x\in A\cup C)\land (x\in B\cup C)\\ \Longleftrightarrow\,\,& x\in (A\cup C)\cap (B\cup C).\end{align*}$

補足

集合の交換法則 [#52] と組み合わせることにより，右側に括弧がある場合：

$\begin{align*} A\cup (B\cap C) &=(A\cup B)\cap (A\cup C),\\
A\cap (B\cup C) &=(A\cap B)\cup (A\cap C).\end{align*}$

の成立も確認できる．



[#56]命題：集合の包含と和

集合 $A,\,B$ が包含関係 $A\subset B$ を持つとき，

$A\cup B=B.$

証明

[#88] より $[x\in B\Rightarrow x\in A\cup B]$ である．仮定より $[x\in A\Rightarrow x\in B]$ であるから，

$\begin{align*} x\in A\cup B\,\,&\Longleftrightarrow\,\, \underline{(x\in A)}\lor (x\in B)\\ &\,\Longrightarrow\,\, \underline{(x\in B)}\lor (x\in B)\,\,\Longleftrightarrow\,\, x\in B.\end{align*}$



[#57]命題：集合の包含と共通部分

集合 $A,\,B$ が包含関係 $A\subset B$ を持つとき，

$A\cap B=A.$

証明

[#88] より $[x\in A\cap B\Rightarrow x\in A]$ である．仮定より $[x\in A\Rightarrow x\in B]$ であるから，

$x\in A\,\,\Longrightarrow\,\, (x\in A)\land (x\in B)\,\,\Longleftrightarrow\,\, x\in A\cap B.$
