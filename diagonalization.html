[#298] 定義：行列の対角化

ある $n$ 次正方行列 $A$ が【対角化可能 (diagonalizable)】であるとは，$A$ に対し正則行列$\nr{regular}{163}$ $P$ と数 $\lambda_1,\ \lambda_2,\ \cdots,\ \lambda_n$ が存在して

\begin{align} \varLambda := P^{-1}AP = \diag (\lambda_1,\,\lambda_2,\,\cdots,\,\lambda_n) \end{align}

という対角行列$\nr{diagonal-matrix}{286}$ を作れることをいう．また，$P$ や $\varLambda$ を求める操作を【対角化 (diagonalization)】と呼ぶ．


 

[#299] 命題：行列の対角化と固有値・固有ベクトル

ある $n$ 次正方行列 $A$ が，列ベクトル $\bs{p}_1,\ \bs{p}_2,\ \cdots,\ \bs{p}_n$ による正規行列 $P = [\bs{p}_1\ \ \bs{p}_2\ \ \cdots\ \ \bs{p}_n]$ で対角化可能ならば，得られた対角行列の各対角成分 $\lambda_i\ (i = 1,\ \cdots,\ n)$ は $A$ の固有値であり，各 $\bs{p}_i$ は $\lambda_i$ に属する固有ベクトルである．

証明

仮定より $A$ は対角化可能$\nr{diagonalization}{298}$ であるから，対角行列 $\varLambda = \diag (\lambda_1,\,\cdots,\,\lambda_n)$ を用いて

\begin{align} \varLambda = P^{-1}AP \end{align}

と表される．この式の両辺に $P$ を左からかけて整理すると

\begin{align} \mathrm{(LHS)} &= P\varLambda = [\lambda_1\bs{p}_1\ \ \cdots\ \ \lambda_n\bs{p}_n],\\ \mathrm{(RHS)} &= PP^{-1}AP \,\refeq{regular}{163}\, I_nAP = AP = [A\bs{p}_1\ \ \cdots\ \ A\bs{p}_n]. \end{align}

を得る（$I_n$ は $n$ 次の単位行列）．故に，各 $i=1,\ \cdots,\ n$ に対して

\begin{align} A\bs{p}_i = \lambda_i\bs{p}_i \end{align}

が成立する．これは，$\lambda_i,\ \bs{p}_i$ がそれぞれ $A$ の固有値とそれに属する固有ベクトルであることを意味する$\nr{eigen}{160}$．

[#301] 命題：行列の対角化による行列の累乗

ある $n$ 次正方行列 $A$ が正規行列 $P$ で対角化できるとして，得られる対角行列を $\varLambda$ とする．このとき，$m\in\mathbb{N}$ に対して

\begin{align} A^m = P\varLambda^mP^{-1}. \end{align}

証明

仮定より

\begin{align} \varLambda = P^{-1}AP \end{align}

と表される$\nr{diagonalization}{298}$．この両辺を $m$ 乗すると

\begin{align} \mathrm{(LHS)} &\,= \varLambda^m,\\ \mathrm{(RHS)} &\,= (P^{-1}AP)^m = (P^{-1}AP)(P^{-1}AP)\cdots(P^{-1}AP)\\ &\refeq{regular}{163} P^{-1}AI_n AI_n\cdots AI_n AP\\ &\refeq{matrix}{185} P^{-1}AA\cdots AP = P^{-1}A^mP. \end{align}

つまり，

\begin{align} \varLambda^m = P^{-1}A^mP. \end{align}

この両辺に左から $P$，右から $P^{-1}$ をかけることで

\begin{align} P\varLambda^m P^{-1} = A^m \end{align}

を得る．両辺を入れ替えれば，それが所望の式である．
