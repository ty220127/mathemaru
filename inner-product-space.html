MATHEMARU

体 $\mathbb{K}$ 上の線形空間 $V$ で，以下の条件を満たす関数 $\braket{\cdot}{\cdot}: V\times V\to \mathbb{K}$ が定義されているものを【内積空間 (inner product space)】，【計量線形空間 (metric linear space)】，或いは【前 Hilbert 空間 (pre-Hilbert space)】と呼ぶ：

  • 【共役対称性 (conjugate symmetry)】

    \begin{align} {}^\forall x,\,y\in V,\ \braket{x}{y} = \overline{\braket{y}{x}}. \end{align}

  • 【線形性 (linearity)】

    \begin{align} {}^\forall a,\ b\in \mathbb{K},\ {}^\forall x,\,y,\,z\in V,\ \braket{ax+by}{z} = a\braket{x}{z} + b\braket{y}{z}. \end{align}

  • 【正定値性 (positive definiteness)】

    \begin{align} {}^\forall x\in V,\ (\braket{x}{x}\geq 0)\land (\braket{x}{x} = 0 \to x = 0) \end{align}

この関数 $\braket{\cdot}{\cdot}$ を【内積 (inner product)】と呼ぶ．

任意の $k,\ \ell\in \mathbb{K}$ と 任意の $x,\ y,\ z\in V$ に対して

\begin{align} \braket{x}{ky + \ell z} = \bar{k}\braket{x}{y} + \bar{\ell}\braket{x}{z} \end{align}

証明

\begin{align} \braket{x}{ky + \ell z} &= \overline{\braket{ky + \ell z}{x}} = \overline{k\braket{y}{x} + \ell\braket{z}{x}}\\ &= \bar{k}\overline{\braket{y}{x}} + \bar{\ell}\overline{\braket{z}{x}} = \bar{k}\braket{x}{y} + \bar{\ell}\braket{x}{z}. \end{align}

補足

このような性質を【反線形性 (anti-linearlity)】或いは【共役線形性 (conjugate linearity)】と呼ぶ．通常の線形性との違いは，定数を関数の外に出すとその定数が元の共役になることである．


 

 

内積空間 $V$ において，関数 $\|\cdot\|: V\to [0,\ \infty)$ を

\begin{align} \|x\| = \sqrt{\braket{x}{x}} \end{align}

により定めると，これはノルムである．
