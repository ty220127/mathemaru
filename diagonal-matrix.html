[#286] 定義：対角行列

非対角成分$\nr{matrix}{144}$ がすべて $0$ である $n$ 次正方行列 $D$ を $n$ 次【対角行列 (diagonal matrix)】と呼ぶ：

$\begin{align*} {}^\forall (i,\,j)\in [n]^2,\quad i\not =j \ \Longrightarrow\ (D)_{ij} = 0. \end{align*}$

$i\in [n]$ に対して $(D)_{ii} = d_i$ とすれば，対角行列 $D$ は

$\begin{align*} D= \left[\begin{array}{cccc} d_{1} & 0 & \cdots & 0\\ 0 & d_{2} & \cdots & 0\\ \vdots & \vdots & \ddots & \vdots\\ 0 & 0 & \cdots & d_{n} \end{array}\right] \end{align*}$

と書ける．また，このような $D$ を

$\begin{align*} D = \diag (d_1,\, d_2,\, \cdots,\, d_n) \end{align*}$

と書くことがある．

補足

すべての $i\in [n]$ に対して $d_i = 1$ であることは，$D$ が単位行列$\nr{matrix}{184}$ $I_n$ であることと同値である．


 

[#287] 命題：対角行列の和・定数倍・積も対角行列

$C,\ D$ を $n$ 次対角行列

$\begin{align*} C = \diag(c_1,\,c_2,\,\cdots,\,c_n),\quad D = \diag(d_1,\,d_2,\,\cdots,\,d_n) \end{align*}$

とする．このとき，以下が成立する：

  1. $C+D = \diag(c_1+d_1,\, c_2+d_2,\, \cdots,\, c_n+d_n)$.
  2. $kD = \diag(kd_1,\, kd_2,\, \cdots,\, kd_n)$ （$k$ は数）.
  3. $CD = DC = \diag(c_1d_1,\, c_2d_2,\, \cdots,\, c_nd_n)$.
証明
  1. 行列の和の定義$\nr{matrix}{145}$ より，$(i,\,j) \in [n]^2$ に対して

    \begin{align} (C+D)_{ij} = (C)_{ij} + (D)_{ij}. \end{align}

    上式は，$i\not =j$ に対して

    \begin{align} (C)_{ij} + (D)_{ij} \,\refeq{diagonal-matrix}{286}\, 0 + 0 = 0, \end{align}

    $i =j$ に対して

    \begin{align} (C)_{ii} + (D)_{ii} \,\refeq{diagonal-matrix}{286}\, c_i + d_i. \end{align}

    と書き換えられるので，題意が成り立つ．

  2. 行列の定数倍の定義$\nr{matrix}{145}$ より，$(i,\,j) \in [n]^2$ に対して

    \begin{align} (kD)_{ij} = k(D)_{ij}. \end{align}

    上式は，$i\not =j$ に対して

    \begin{align} k(D)_{ij} \,\refeq{diagonal-matrix}{286}\, k\cdot 0 = 0, \end{align}

    $i =j$ に対して

    \begin{align} k(D)_{ii} \,\refeq{diagonal-matrix}{286}\, kd_i. \end{align}

    と書き換えられるので，題意が成り立つ．

  3. $CD = \diag(c_1d_1,\, c_2d_2,\, \cdots,\, c_nd_n)$ を示す．$DC$ の場合も同様である．
    行列の積の定義$\nr{matrix}{145}$ より，$(i,\,j) \in [n]^2$ に対して

    \begin{align} (CD)_{ij} = \sum_{k=1}^{n} (C)_{ik}(D)_{kj}. \end{align}

    ここで，$i=k=j$ のとき

    \begin{align} (C)_{ik}(D)_{kj} \,\refeq{diagonal-matrix}{286}\, c_i d_i \end{align}

    が成り立つが，そうでないときは

    \begin{align} (i\not =k)\lor (k\not =j)\ \reflr{diagonal-matrix}{286}\ ((C)_{ik}=0)\lor ((D)_{kj}\not =0) \end{align}

    であることから $(C)_{ik}(D)_{kj}=0$ である．よって

    \begin{align} \sum_{k=1}^{n} (C)_{ik}(D)_{kj} = \left\{\begin{array}{ll} c_id_i & (i=j)\\ 0 & (\text{otherwise}) \end{array}\right. \end{align}

    であることから，題意は示される．


 

[#288] 命題：対角行列の $\bs{m}$ 乗は各成分の $\bs{m}$ 乗

$D$ を $n$ 次対角行列

$\begin{align*} D = \diag(d_1,\,d_2,\,\cdots,\,d_n) \end{align*}$

とすると，任意の $m\in\mathbb{N}$ に対して

$\begin{align*} D^m = \diag(d_1^m,\,d_2^m,\,\cdots,\,d_n^m) \end{align*}$

証明

数学的帰納法により示す．$m=1$ のときは自明．ある $m$ で題意が成立すると仮定する．このとき，

\begin{align*} D^{m+1} &\,= D^m D\\ &\,= \diag(d_1^m,\,d_2^m,\,\cdots,\,d_n^m)\cdot\diag(d_1,\,d_2,\,\cdots,\,d_n)\\ &\refeq{diagonal-matrix}{287}\,\diag(d_1^{m+1},\,d_2^{m+1},\,\cdots,\,d_n^{m+1}). \end{align*}

よって，$m+1$ でも題意は成立するので，任意の $m$ に関して題意は成立する．

$D^0 = I_n$ (単位行列) と定義すれば，$m\in\mathbb{N}_0 = \mathbb{N}\cup \{0\}$ まで拡張できる．
