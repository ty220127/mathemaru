[#95] 定義：差集合

集合 $A$ に属し $B$ に属さないような要素の集合を $A$ から $B$ の【差集合(set difference)】と呼び，$A\setminus B$ と書く：

$\begin{align*}A\setminus B &= \{x\,|\,\,(x\in A) \land (x\notin B)\}\\ &= \{x\,|\,\,(x\in A) \land (\neg(x\in B))\}.\end{align*}$


 

[#85] 命題：補集合を用いた差集合の表現

集合 $A,\,B$ に対して

$A\setminus B=A\cap B^\cpl.$

特に，普遍集合 $\varOmega$ に対して

$\varOmega\setminus B= B^\cpl.$

証明

\begin{align*} x\in A\setminus B\,\,\reflr{difference-set}{95}\,\,& (x\in A)\land (x\notin B)\\ \reflr{complement-set}{84}\,\,& (x\in A)\land (x\in B^\cpl)\\ \reflr{set}{51}\,\,& x\in A\cap B^\cpl. \end{align*}

よって $A\setminus B=A\cap B^\cpl$ が従う．
ここで $A$ に普遍集合 $\varOmega$ を代入すると

\begin{align} \mathrm{(LHS)} &= \varOmega\setminus B,\\ \mathrm{(RHS)} &= \varOmega\cap B^\cpl \ \refeq{universal-set}{215}\ B^\cpl.\\ \end{align}

結果をまとめると，$\varOmega\setminus B = B^\cpl.$


 

[#211] 命題：差集合と空集合

集合 $A$ に対して

$A\setminus \emptyset = A,\quad\emptyset\setminus A = \emptyset,\quad A\setminus A = \emptyset.$

証明１（補集合を用いる）

\begin{align} A\setminus \emptyset \ \refeq{difference-set}{85}\ A\cap \emptyset^\cpl\ \refeq{universal-set}{213}\ A\cap \varOmega \ \refeq{universal-set}{215}\ A, \end{align}

\begin{align} \emptyset\setminus A \ \refeq{difference-set}{85}\ \emptyset\cap A^\cpl\ \refeq{universal-set}{325}\ \emptyset, \end{align}

\begin{align} A\setminus A \ \refeq{difference-set}{85}\ A\cap A^\cpl\ \refeq{universal-set}{214}\ \emptyset. \end{align}

証明２（定義から直接）

$A\setminus\emptyset =A:$

\begin{align*} x\in A\setminus \emptyset &\,\,\reflr{#95}{95}\,\,(x\in A)\land (\neg(x\in \emptyset))\\ &\,\,\reflr{universal-set}{110}\,\,(x\in A)\land (\neg\ \mathsf{F})\\ &\,\,\Longleftrightarrow\,\,(x\in A)\land \mathsf{T}\,\,\reflr{proposition}{60}\,\,x\in A.\\ \end{align*}

$\emptyset\setminus A =\emptyset:$

\begin{align*} x\in \emptyset\setminus A &\,\,\reflr{difference-set}{95}\,\,(x\in \emptyset)\land (x\not\in A)\\ &\,\,\reflr{universal-set}{110}\,\,\mathsf{F} \land (x\not \in A)\\ &\,\,\reflr{proposition}{60}\,\,\mathsf{F}\,\,\reflr{universal-set}{110}\,\,x\in \emptyset.\\ \end{align*}

$A\setminus A =\emptyset:$

\begin{align*} x\in A\setminus A &\,\,\reflr{difference-set}{95}\,\,(x\in A)\land (\neg(x\in A))\\ &\,\,\reflr{proposition}{60}\,\,\mathsf{F}\,\,\reflr{universal-set}{110}\,\,x\in \emptyset.\\ \end{align*}


 

[#96] 命題：差集合と共通部分

集合 $A,\,B$ に対して

$A \cap (B\setminus A) = \emptyset.$

証明１（補集合を用いる）

\begin{align} A \cap (B\setminus A) \ \refeq{difference-set}{85}\ &A\cap B\cap A^\cpl\ \refeq{set}{52}\ A\cap A^\cpl\cap B \\ \refeq{universal-set}{214}\ & \emptyset\cap B \ \refeq{universal-set}{325}\ \emptyset. \end{align}

証明２（定義から直接）

\begin{align*} \,\,& x\in A \cap (B\setminus A)\\ \reflr{defference-set}{95}\ & (x\in A)\land (x\in B)\land (\neg(x\in A))\\ \reflr{proposition}{60}\ & \mathsf{F}\land (x\in B) \ \reflr{proposition}{60}\ \mathsf{F}. \end{align*}

よって $A \cap (B\setminus A) = \emptyset$ である$\nr{universal-set}{110}$．


 

[#97] 命題：差集合と和

集合 $A,\,B$ に対して

$A \cup (B\setminus A)= A\cup B.$

特に $A\subset B$ であるならば

$A \cup (B\setminus A) = B.$

証明１（補集合を用いる）

\begin{align} A \cup (B\setminus A) \ \refeq{difference-set}{85}\ &A\cup (B\cap A^\cpl)\\ \refeq{set}{54}\ &(A\cup B)\cap (A\cap A^\cpl)\\ \refeq{universal-set}{214}\ &(A\cup B)\cap \varOmega \ \refeq{universal-set}{215}\ A\cup B.\\ \end{align}

特に $A\subset B$ であるならば

$A \cup (B\setminus A) = A\cup B \ \refeq{set}{56}\ B.$

証明２（定義から直接）

$\begin{align*} \,\,& x\in A \cup (B\setminus A)\\ \reflr{defference-set}{95}\ & (x\in A)\lor ((x\in B)\land \neg(x\in A))\\ \reflr{proposition}{62}\ & ((x\in A)\lor (x\in A))\land ((x\in A)\lor \neg(x\in A))\\ \reflr{proposition}{60}\ & (x\in A\cup B)\land \mathsf{T}\\ \reflr{proposition}{60}\ & x\in A\cup B.\end{align*}$

よって $A \cup (B\setminus A)= A\cup B$ が成り立つ．


 

[#55] 命題：直和による集合の分解

集合 $A,\,B$ に対して，$A\cap B$ と $A\setminus B$ は互いに素であり，

$\begin{align*} (A\cap B)\sqcup(A\setminus B) = A. \end{align*}$

証明１（補集合を用いる）

まず，

\begin{align*} (A\cap B)\cap (A\setminus B) \ \refeq{set}{53}\ A\cap \{B\cap (A\setminus B)\} \ \refeq{difference-set}{96}\ A\cap \emptyset \ \refeq{universal-set}{325}\ \emptyset \end{align*}

より $A\cap B$ と $A\setminus B$ は互いに素である．また，これらの和（直和）について，

\begin{align} (A\cap B)\cup (A\setminus B) \ \refeq{difference-set}{95}\ &(A\cap B)\cup (A\cap B^\cpl)\\ \ \refeq{set}{54}\ &A\cap (B\cup B^\cpl) \ \refeq{universal-set}{214}\ A\cap \varOmega \ \refeq{universal-set}{215}\ A. \end{align}

証明２（定義から直接）

$\begin{align*} \,\,& x\in (A\cap B)\cup (A\setminus B)\,\,\reflr{set.html#51}{51}\,\, (x\in A\cap B) \lor (x\in (A\setminus B))\\ \reflr{#95}{95}\,\,& ((x\in A) \land (x\in B)) \lor ((x\in A) \land (x\notin B))\\ \reflr{proposition.html#62}{62}\,\,& (x\in A) \land ((x\in B) \lor (x\notin B))\\ \reflr{proposition.html#58}{58}\,\,& (x\in A) \land \mathsf{T}\,\,\reflr{proposition.html#60}{60}\,\, x\in A.\end{align*}$

よって $(A\cap B)\sqcup(A\setminus B) = A$ がいえる．


 

[#212] 命題：差集合の分配法則

集合 $A,\,B,\,C$ に対して

$\begin{align*} (A\cup B)\setminus C = (A\setminus C)\cup (B\setminus C),\\ (A\cap B)\setminus C = (A\setminus C)\cap (B\setminus C). \end{align*}$

証明１（補集合を用いる）

\begin{align} (A\cup B)\setminus C \ \refeq{difference-set}{85}\ &(A\cup B)\cap C^\cpl \\ \ \refeq{set}{54}\ &(A\cap C^\cpl)\cup (B\cap C^\cpl) \\ \ \refeq{difference-set}{85}\ &(A\setminus C)\cup (B\setminus C), \end{align}

\begin{align} (A\cap B)\setminus C \ \refeq{difference-set}{85}\ &(A\cap B)\cap C^\cpl \\ \ \refeq{set}{52}\ &(A\cap C^\cpl)\cap (B\cap C^\cpl) \\ \ \refeq{difference-set}{85}\ &(A\setminus C)\cap (B\setminus C). \end{align}

証明２（定義から直接）

$(A\cup B)\setminus C = (A\setminus C)\cup (B\setminus C):$

\begin{align*} x\in (A\cup B)\setminus C &\,\,\reflr{#95}{95}\,\,(x\in A\cup B)\land (x\not\in C)\\ &\,\,\reflr{set.html#51}{51}\,\,\{(x\in A)\lor (x\in B)\}\ \land (x\not\in C)\\ &\,\,\reflr{proposition.html#62}{62}\,\,\{(x\in A)\land (x\not\in C)\}\lor \{(x\in B)\land (x\not\in C)\}\\ &\,\,\reflr{#95}{95}\,\,(x\in A\setminus C)\lor (x\in B\setminus C)\\ &\,\,\reflr{set.html#51}{51}\,\,x\in (A\setminus C)\cup (B\setminus C). \end{align*}

$(A\cap B)\setminus C = (A\setminus C)\cap (B\setminus C):$

\begin{align*} x\in (A\cap B)\setminus C &\,\,\reflr{#95}{95}\,\,(x\in A\cap B)\land (x\not\in C)\\ &\,\,\reflr{set.html#51}{51}\,\,\{(x\in A)\land (x\in B)\}\ \land (x\not\in C)\\ &\,\,\Longleftrightarrow\,\,\{(x\in A)\land (x\not\in C)\}\lor \{(x\in B)\land (x\not\in C)\}\\ &\,\,\reflr{#95}{95}\,\,(x\in A\setminus C)\land (x\in B\setminus C)\\ &\,\,\reflr{set.html#51}{51}\,\,x\in (A\setminus C)\cap (B\setminus C). \end{align*}


 

[#326] 命題：de Morgan の法則と差集合

集合 $A,\,B,\,C$ に対して

$\begin{align*} C\setminus (A\cup B)= (C\setminus A)\cap (C\setminus B),\\ C\setminus (A\cap B)= (C\setminus A)\cup (C\setminus B),\\ \end{align*}$

証明

\begin{align} C\setminus (A\cup B) \ \refeq{difference-set}{85}\ &C\cap (A\cup B)^\cpl \ \refeq{complement-set}{74}\ C\cap A^\cpl \cap B^\cpl\\ =\ & C\cap C\cap A^\cpl \cap B^\cpl \ \refeq{set}{52}\ C\cap A^\cpl\cap C \cap B^\cpl\\ \refeq{difference-set}{85}\ &(C\setminus A)\cap (C\setminus B), \end{align}

\begin{align} C\setminus (A\cap B) \ \refeq{difference-set}{85}\ &C\cap (A\cap B)^\cpl \ \refeq{complement-set}{74}\ C\cap (A^\cpl \cup B^\cpl)\\ \refeq{set}{54}\ & (C\cap A^\cpl) \cup (C\cap B^\cpl)\ \refeq{difference-set}{85}\ (C\setminus A)\cup (C\setminus B), \end{align}
