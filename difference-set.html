[#95] 定義：差集合

集合 $A$ に属し $B$ に属さないような要素の集合を $A$ から $B$ の【差集合(set difference)】と呼び，$A\setminus B$ と書く：

$\begin{align*}A\setminus B &= \{x\,|\,\,(x\in A) \land (x\notin B)\}\\ &= \{x\,|\,\,(x\in A) \land (\neg(x\in B))\}.\end{align*}$



[#211] 命題：差集合と空集合

集合 $A$ に対して

$A\setminus \emptyset = A,\quad\emptyset\setminus A = \emptyset,\quad A\setminus A = \emptyset.$

証明

$A\setminus\emptyset =A:$

\begin{align*} x\in A\setminus \emptyset &\,\,\reflr{#95}{95}\,\,(x\in A)\land (\neg(x\in \emptyset))\\ &\,\,\reflr{set.html#110}{110}\,\,(x\in A)\land (\neg\ \mathsf{F})\\ &\,\,\Longleftrightarrow\,\,(x\in A)\land \mathsf{T}\,\,\reflr{proposition.html#60}{60}\,\,x\in A.\\ \end{align*}

$\emptyset\setminus A =\emptyset:$

\begin{align*} x\in \emptyset\setminus A &\,\,\reflr{#95}{95}\,\,(x\in \emptyset)\land (x\not\in A)\\ &\,\,\reflr{set.html#110}{110}\,\,\mathsf{F} \land (x\not \in A)\\ &\,\,\reflr{proposition.html#60}{60}\,\,\mathsf{F}\,\,\reflr{set.html#110}{110}\,\,x\in \emptyset.\\ \end{align*}

$A\setminus A =\emptyset:$

\begin{align*} x\in A\setminus A &\,\,\reflr{#95}{95}\,\,(x\in A)\land (\neg(x\in A))\\ &\,\,\reflr{proposition.html#60}{60}\,\,\mathsf{F}\,\,\reflr{set.html#110}{110}\,\,x\in \emptyset.\\ \end{align*}



[#96] 命題：差集合と共通部分

集合 $A,\,B$ に対して

$A \cap (B\setminus A) = \emptyset.$

証明

$\begin{align*} \,\,& x\in A \cap (B\setminus A)\\ \Longleftrightarrow\,\,& (x\in A)\land (x\in B)\land (\neg(x\in A))\\ \Longleftrightarrow\,\,& \mathsf{F}\land (x\in B) \equiv \mathsf{F}.\end{align*}$

よって空集合の定義により，上の $x$ について $x\in\emptyset$．



[#97] 命題：差集合と和

集合 $A,\,B$ に対して

$A \cup (B\setminus A) = A\cup B.$

特に $A\subset B$ であるならば

$A \cup (B\setminus A) = B.$

証明

$\begin{align*} \,\,& x\in A \cup (B\setminus A)\\ \underset{\href{#95}{\sf[\# 95]}}{\Longleftrightarrow}\,\,& (x\in A)\lor ((x\in B)\land \neg(x\in A))\\ \underset{\href{proposition.html#62}{\sf[\# 62]}}{\Longleftrightarrow}\,\,& ((x\in A)\lor (x\in A))\land ((x\in A)\lor \neg(x\in A))\\ \underset{\href{proposition.html#60}{\sf[\# 60]}}{\Longleftrightarrow}\,\,& (x\in A\cup B)\land \mathsf{T}\\ \underset{\href{proposition.html#60}{\sf[\# 60]}}{\Longleftrightarrow}\,\,& x\in A\cup B.\end{align*}$

特に $A\subset B$ であるならば [#56] より

$A \cup (B\setminus A) = A\cup B = B.$



[#55] 命題：直和による集合の分解

集合 $A,\,B$ に対して，$A\cap B$ と $A\setminus B$ は互いに素であり，

$\begin{align*} (A\cap B)\sqcup(A\setminus B) = A. \end{align*}$

証明

まず，

\begin{align*} (A\cap B)\cap (A\setminus B) \refeq{set.html#53}{53} A\cap \{B\cap (A\setminus B)\} \refeq{#96}{96} A\cap \emptyset = \emptyset \end{align*}

より $A\cap B$ と $A\setminus B$ は互いに素である．また，これらの和（直和）について，

$\begin{align*} \,\,& x\in (A\cap B)\cup (A\setminus B)\,\,\reflr{set.html#51}{51}\,\, (x\in A\cap B) \lor (x\in (A\setminus B))\\ \reflr{#95}{95}\,\,& ((x\in A) \land (x\in B)) \lor ((x\in A) \land (x\notin B))\\ \reflr{proposition.html#62}{62}\,\,& (x\in A) \land ((x\in B) \lor (x\notin B))\\ \reflr{proposition.html#58}{58}\,\,& (x\in A) \land \mathsf{T}\,\,\reflr{proposition.html#60}{60}\,\, x\in A.\end{align*}$

であることから $(A\cap B)\sqcup(A\setminus B) = A$ がいえる．



[#212] 命題：差集合の分配法則

集合 $A,\,B,\,C$ に対して

$\begin{align*} (A\cup B)\setminus C = (A\setminus C)\cup (B\setminus C),\\ (A\cap B)\setminus C = (A\setminus C)\cap (B\setminus C). \end{align*}$

証明

$(A\cup B)\setminus C = (A\setminus C)\cup (B\setminus C):$

\begin{align*} x\in (A\cup B)\setminus C &\,\,\reflr{#95}{95}\,\,(x\in A\cup B)\land (x\not\in C)\\ &\,\,\reflr{set.html#51}{51}\,\,\{(x\in A)\lor (x\in B)\}\ \land (x\not\in C)\\ &\,\,\reflr{proposition.html#62}{62}\,\,\{(x\in A)\land (x\not\in C)\}\lor \{(x\in B)\land (x\not\in C)\}\\ &\,\,\reflr{#95}{95}\,\,(x\in A\setminus C)\lor (x\in B\setminus C)\\ &\,\,\reflr{set.html#51}{51}\,\,x\in (A\setminus C)\cup (B\setminus C). \end{align*}

$(A\cap B)\setminus C = (A\setminus C)\cap (B\setminus C):$

\begin{align*} x\in (A\cap B)\setminus C &\,\,\reflr{#95}{95}\,\,(x\in A\cap B)\land (x\not\in C)\\ &\,\,\reflr{set.html#51}{51}\,\,\{(x\in A)\land (x\in B)\}\ \land (x\not\in C)\\ &\,\,\Longleftrightarrow\,\,\{(x\in A)\land (x\not\in C)\}\lor \{(x\in B)\land (x\not\in C)\}\\ &\,\,\reflr{#95}{95}\,\,(x\in A\setminus C)\land (x\in B\setminus C)\\ &\,\,\reflr{set.html#51}{51}\,\,x\in (A\setminus C)\cap (B\setminus C). \end{align*}
