[#310] 定義：Bernoulli 分布

確率変数$\nr{random-variable}{23}$ $X$ の分布$\nr{random-variable}{173}$ が以下で与えられることを，$X$ は【Bernoulli(ベルヌーイ) 分布 (Bernoulli distribution)】に従うという： $0\leq p\leq 1$ に対して，

\begin{align} P(X = k) = p^k(1-p)^{1-k}\quad (k = 0,\,1). \end{align}

この $X$ を【Bernoulli 確率変数 (Bernoulli random variable)】と呼ぶことがある．またパラメータ $p$ の Bernoulli 分布を $\mathrm{Ber}(p)$ と書く．

補足

$k$ にそれぞれ $0$ と $1$ を代入してみれば，確率 $1-p$ で $X=0$，$p$ で $X=1$ となる分布であることがわかる．この $0$ と $1$ をコインの裏表や試行の失敗／成功など，結果が２通りである試行に充ててモデル化を行う例が多い．


 

[#311] 命題：Bernoulli 分布の期待値

確率変数$\nr{random-variable}{23}$ $X$ がパラメータ $p$ の Bernoulli 分布に従うとき，その期待値は

\begin{align} \mathbb{E}[X] = p. \end{align}

証明

\begin{align} \mathbb{E}[X] = 0\cdot (1-p) + 1\cdot p = p. \end{align}


 

[#312] 命題：Bernoulli 分布の分散

確率変数$\nr{random-variable}{23}$ $X$ がパラメータ $p$ の Bernoulli 分布に従うとき，その分散は

\begin{align} \mathbb{V}[X] = p(1-p). \end{align}

証明

\begin{align} \mathbb{V}[X] &= \mathbb{E}[X^2] -(\mathbb{E}[X])^2 \,\refeq{bernoulli-distribution}{311}\, \mathbb{E}[X^2] -p^2. \end{align}

ここで

\begin{align} \mathbb{E}[X^2] = 0^2\cdot (1-p) + 1^2\cdot p = p \end{align}

であるから，

\begin{align} \mathbb{E}[X^2] -p^2 = p-p^2 = p(1-p). \end{align}

以上の議論をまとめることで，題意を得る．


 

[#313] 演習：複数回のコイントス → $\numref{binomial-distribution}{315}$ $\numref{binomial-distribution}{316}$ $\numref{binomial-distribution}{317}$

裏表の出る確率が等しいコインのトスを $n$ 回繰り返し，表の出る回数を調べる．各コイントスはすべて独立とし，$n$ 回目のトスに対応する確率変数を $X_n\sim \mathrm{Ber}(1/2)$ とする．

  1. コイントスを $3$ 回行ってすべて表が出る確率を求めよ．
  2. コイントスを $5$ 回行って $3$ 回表が出る確率を求めよ．
  3. コイントスを $5$ 回行ったときの，表が出た回数の期待値と分散を求めよ．
解答

以下，表を $1$，裏を $0$ で表す．

  1. この事象は $X_1 = X_2 =X_3 =1$ なる場合に対応し，その確率は

    \begin{align} &P((X_1 = 1)\land (X_2 =1)\land (X_3 =1)) \\ \,\refeq{independence}{209}\,&P(X_1=1)P(X_2=1)P(X_3=1) = \color{red}{\fraction{1}{8}}. \end{align}

  2. 1，2回目が表，3回目以降はすべて裏の場合を考える．

    \begin{align} &P((X_1 = 1)\land (X_2 =1)\land (X_3 =0)\land (X_4 =0)\land (X_4 =0)) \\ \,\refeq{independence}{209}\,&P(X_1=1)P(X_2=1)P(X_3=0)P(X_4=0)P(X_4=0) = \fraction{1}{32}. \end{align}

    3. 表の出る回を別の回に変えても明らかに確率は同じである．表の出る回の組合せは $\left(\begin{array}{c}5\\ 2\end{array}\right) =10$ (通り) であるから，求める確率は

    \begin{align} \fraction{1}{32}\times 10 = \color{red}{\fraction{5}{16}}. \end{align}

  3. コイントスを $5$ 回行ったときに表の出た回数は，

    \begin{align} S = \sum_{k=1}^{5} X_k \end{align}

    で表される．$S$ の期待値と分散は

    \begin{align} &\mathbb{E}[S] = \sum_{k=1}^{5}\mathbb{E}[X_k] = \sum_{k=1}^{5}\fraction{1}{2} = \color{red}{\fraction{5}{2}},\\ &\mathbb{V}[S] = \sum_{k=1}^{5}\mathbb{V}[X_k] = \sum_{k=1}^{5} \left(\fraction{1}{2}\cdot \fraction{1}{2}\right) = \color{red}{\fraction{5}{4}}. \end{align}
