[#310] 定義：Bernoulli 分布

確率変数$\nr{random-variable}{23}$ $X$ の分布$\nr{random-variable}{173}$ が以下で与えられることを，$X$ は【Bernoulli(ベルヌーイ) 分布 (Bernoulli distribution)】に従うという：

\begin{align} P(X = k) = p^k(1-p)^{1-k}\quad (k = 0,\,1). \end{align}

$X$ がパラメータ $p$ の Bernoulli 分布に従うことを $X\sim \mathrm{Ber}(p)$ と書くことがある．

補足

$k$ にそれぞれ $0$ と $1$ を代入してみれば，確率 $1-p$ で $X=0$，$p$ で $X=1$ となる分布であることがわかる．この $0$ と $1$ をコインの表裏や試行の失敗／成功に充てて考える例が多い．


 

[#311] 命題：Bernoulli 分布の期待値

確率変数$\nr{random-variable}{23}$ $X$ がパラメータ $p$ の Bernoulli 分布に従うとき，その期待値は

\begin{align} \mathbb{E}[X] = p. \end{align}

証明

\begin{align} \mathbb{E}[X] = 0\cdot (1-p) + 1\cdot p = p. \end{align}


 

[#312] 命題：Bernoulli 分布の分散

確率変数$\nr{random-variable}{23}$ $X$ がパラメータ $p$ の Bernoulli 分布に従うとき，その分散は

\begin{align} \mathbb{V}[X] = p(1-p). \end{align}

証明

\begin{align} \mathbb{V}[X] &= \mathbb{E}[X^2] -(\mathbb{E}[X])^2 \,\refeq{bernoulli-distribution}{311}\, \mathbb{E}[X^2] -p^2. \end{align}

ここで

\begin{align} \mathbb{E}[X^2] = 0^2\cdot (1-p) + 1^2\cdot p = p \end{align}

であるから，

\begin{align} \mathbb{E}[X^2] -p^2 = p-p^2 = p(1-p). \end{align}

以上の議論をまとめることで，題意を得る．
