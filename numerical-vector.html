[#432] 定義：数ベクトル，数ベクトル空間

$n$ 個の体 $\mathbb{K}$ による直積

\begin{align} \mathbb{K}^n = \{(v_1,\,v_2,\,\cdots,\,v_n)\,|\,{}^\forall i,\ v_i\in \mathbb{K}^n\} \end{align}

で，$\bs{u} = (u_1,\,u_2,\,\cdots,\,u_n),\ \bs{v} = (v_1,\,v_2,\,\cdots,\,v_n)\in \mathbb{K}^n$ と $k\in \mathbb{K}$ に対して以下のような演算が定義されているとする：

\begin{align} \bs{u} + \bs{v} &:= (u_1+v_1,\,u_2+v_2,\,\cdots,\,u_n+v_n),\\ k\bs{v} &:= (k v_1,\,k v_2,\,\cdots,\,k v_n). \end{align}

この $\mathbb{K}^n$ を $\mathbb{K}$ 上の $n$ 次元【数ベクトル空間 (numerical vector space)】或いは単に【ベクトル空間 (vector space)】と呼び，上記演算を順に【加法 (addition)】，【スカラー倍 (scalar product)】と呼ぶ．また，加法の結果を $\bs{u}$ と $\bs{v}$ の【和 (sum)】と呼ぶ．

$\mathbb{K}^n$ の元を $\mathbb{K}$ 上の $n$ 次元【数ベクトル (numerical vector)】或いは単に【ベクトル (vector)】と呼ぶ．また，ベクトル $\bs{v} = (v_1,\,v_2,\,\cdots,\,v_n)$ に対して各 $v_i\,(i = 1,\,2,\,\cdots,\,n)$ を $\bs{v}$ の【成分 (element)】と呼び，特に $v_i$ を $\bs{v}$ の【第 $i$ 成分 ($i$-th element)】と呼ぶ．

補足

数ベクトルは上記のように太字で表すことがあるほか，$\vec{v}$ などと矢印を付記することで表すこともある．


 

[#433] 定義：逆ベクトル，ベクトルの減法

体 $\mathbb{K}$ 上の $n$ 次元ベクトル $\bs{v} = (v_1,\,v_2,\,\cdots,\,v_n)\in \mathbb{K}^n$ に対して，

\begin{align} (-1)\bs{v} \,\refeq{numerical-vector}{432}\, (-v_1,\,-v_2,\,\cdots,\,-v_n) \end{align}

を $\bs{v}$ の【逆ベクトル (inverse vector)】と呼び，$-\bs{v}$ で表す．

また，$\bs{u},\ \bs{v}\in\mathbb{K}^n$ に対して，加法

\begin{align} \bs{u} + (-\bs{v}) \,\refeq{numerical-vector}{432}\, (u_1-v_1,\,u_2-v_2,\,\cdots,\,u_n-v_n) \end{align}

を $\bs{u}$ と $\bs{v}$ の【差 (difference)と呼び，$\bs{u}-\bs{v}$ で表す．またこの演算を【減法 (subtraction)】と呼ぶ．


 

[#434] 定義：零ベクトル

体 $\mathbb{K}$ 上の $n$ 次元ベクトルで，すべての成分が $0$ であるものを $\mathbb{K}^n$ 上の【零ベクトル (zero vector)】と呼び，$\mathbf{0},\ \bs{o}$ などで表す：

\begin{align} \mathbf{0} = (0,\,0,\,\cdots,\,0). \end{align}
